# 多媒体工具箱 - 用户使用手册

欢迎使用 **多媒体工具箱**。本软件旨在为您提供一个全能、离线且高度注重隐私的音视频处理环境。所有操作均在您的本地电脑上完成，无需上传云端，确保您的文件绝对安全。

---

## 🎬 核心功能一：全格式互转

此模块可以帮助您在各种视频、音频和图片格式之间进行批量转换，或对媒体文件进行压制（压缩体积）。

### 如何使用：
1. **添加文件**：
   - 您可以直接将媒体文件拖拽到“待处理文件”列表中，也可以点击“➕ 添加文件”按钮浏览选择。
   - 支持批量拖入整个文件夹。
2. **设置输出格式与参数**：
   - **目标格式**：从下拉菜单中选择您需要转换成的格式（例如 MP4、MP3、WEBP）。
   - **编码质量 (CRF)**：滑动条用于调整视频/图片质量。数值越小，画质越好，但文件越大；数值越大，画质越差，但文件越小。**推荐保持默认的 23**。
   - **预设速度**：选择编码速度（如 fast, medium, slow）。速度越慢，压制出来的文件在同等画质下体积更小。
   - **硬件加速**：如果您有独立显卡，可以勾选 NVIDIA (NVENC) 选项，大幅提升视频转换速度。
3. **输出选项**：
   - 默认会自动生成并选择一个“output”文件夹，您也可以点击“选择目录”更改。
   - 勾选“生成覆盖无声视频”可以用来给视频静音。
   - 可以设置“并发任务数”，即同时转换多少个文件。如果您的 CPU 性能较强，可以调高该数值。
4. **开始转换**：点击“🚀 开始批量转换”，底部的进度条和日志窗口会实时显示处理进度。

## 🛡️ 核心功能二：元数据隐私清理

当我们用手机拍摄照片或视频时，文件中往往会隐藏大量的**隐私元数据**（如：GPS 定位、拍摄时间、手机型号、甚至系统账号信息）。直接将这些原文件发给他人或传到网上，可能会泄露您的个人隐私。此模块专门用来彻底剥离这些隐藏信息。

### 如何使用：
1. **添加文件**：与格式转换一样，拖拽或点击添加文件。
2. **开始清理**：点击“🛡️ 开始并发擦除元数据”。
3. **查看审计报告**：
   - 清理完成后，下方会生成一个“复查统计表”。
   - **原始字段数** vs **清理后字段数**：您可以清楚地看到有多少条隐藏信息被成功剥离。
   - **结果判定**：如果显示“✅ 通过”，说明所有涉及隐私的字段都已消失。
   - *注意*：鼠标悬停在表格的长文本上，可以弹出完整内容。您可能会看到残留了类似 `VideoFrameRate` 或 `major_brand` 等字段，这些是保障视频正常播放所必须的**纯物理结构参数**，绝对不含个人隐私，您可以 100% 放心。
4. **导出审计日志**：如果需要，您可以将清理前后的详细对比数据导出为 JSON 或 CSV 报表。

## 🔎 核心功能三：媒体元信息查看

如果您只是想“侦查”一下某个文件里到底藏了什么信息，而不打算对它做修改，可以使用这个模块。

### 如何使用：
1. 直接将**单个文件**拖入窗口左侧区域（或点击“选择文件”）。
2. **基础属性 (FFprobe)**：展示该文件的容器信息、视频流（分辨率、码率、帧率）和音频流详情。
3. **全量元数据 (ExifTool)**：点击这个标签页，系统会利用底层工具像 X 光一样扫出文件里所有的标签。如果该文件没有被清理过，您往往会在这里看到拍摄它的设备名称甚至地理坐标。

## 常见问题与提示
- **工具依赖**：本软件底层依赖强大的开源引擎 FFmpeg 和 ExifTool。若某些功能受限，请检查程序目录的 `tools/` 文件夹下是否缺少这两个引擎。
- **强制停止**：在处理大量文件时，您可以随时点击“⏹️ 取消”按钮，软件会安全地中断剩余的任务。
- **文件覆盖冲突**：在处理同名文件时，系统会自动在文件名后追加数字后缀，永远不会覆盖或损坏您的原文件。
